# 🚀 RAG + Memory AI Agent with n8n

Built an end-to-end Retrieval-Augmented Generation (RAG) AI Agent using:

- n8n
- Supabase Vector Database
- OpenAI Embeddings
- Claude 3 Haiku via OpenRouter
- PostgreSQL Memory

## Architecture

Dataset
↓
Chunking
↓
OpenAI Embeddings
↓
Supabase Vector Store
↓
Retriever
↓
Claude 3 Haiku
↓
PostgreSQL Memory
↓
Context-Aware Response
